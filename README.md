# Carat-BackEnd-Django

Django 3.1  
django-rest-auth 0.9.5  
djangorestframework 3.11.1  
djangorestframework-jwt 1.11.0  
Pillow 7.2.0  
PyJWT 1.7.1  
PyMySQL 0.10.0
